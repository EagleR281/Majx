EXPLANATION

If you finish setting everything up and want to play, make sure not to close the program. If it's bothering you, just minimize it (−). If you close it, you won’t benefit from it. If your device has a history of issues (such as cheats, scripts, or unsafe/free tools), it’s recommended to open a support ticket and request a system cleanup. This can significantly reduce the risk of a ban, potentially up to 100%.

Input: In-game input must be set to Mouse. The script uses mouse-style movement and sensitivities you can tune inside the script. Mouse input is required in all cases.
